# AP Assembly Elections 2024 Insights

A comprehensive dashboard for analyzing the 2024 Andhra Pradesh Assembly Election results.

## Features

- **Constituency Overview**: Detailed results for each of the 175 constituencies.
- **Party Performance**: High-level analysis of seats won, vote shares, and regional dominance.
- **District breakdown**: Performance analysis aggregated at the district level.
- **Dynamic UI**: interactive charts and tables with premium aesthetics.

## Technologies

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn-ui
- **Icons**: Lucide React
- **Charts**: Recharts
- **SEO**: React Helmet Async

## Getting Started

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Run in development mode**:
   ```sh
   npm run dev
   ```

3. **Build for production**:
   ```sh
   npm run build
   ```

## Customization

- **Logo**: Custom party logos can be placed in `public/logos/` as `{PartyAcronym}.png`.
- **Colors**: Party colors and theme variables are defined in `src/index.css` and `src/lib/partyColors.ts`.
